# Analyse de phénologie de chant des Râles des genêts par acoustique passive

Ce projet Github a été structuré pour pouvoir étudier la phénologie de chant du Râle des genêts (*Crex crex*) à partir de données d'acoustique passive (audiomoth ou SM).
Cette analyse est développée pour un protocole de suivi basé sur 1 minute d'enregistrement par tranche de 10 minutes en continu toute la journée, soit 144 enregistrements par jour. Ces fichiers sont ensuite à analyser sur BirdNET Analyzer avant de pouvoir utiliser le script joint.
L'ensemble des fichiers de paramétrage de BirdNET + des Audiomoths sont contenus dans le dossier "aide_prog"

Dans le dossier "analyses", le fichier "script_analyses.Qmd" contient toutes les étapes pour analyser et explorer les données audio issues des enregistreurs - en format .wav ou .mp3.
Dans le dossier "data", vous retrouverez un exemple de structuration d'un dossier pour la suite des analyses, ainsi que les .csv et  les figures issues du script .Qmd.

Ce Github sera mis à jour régulièrement en fonction d'éventuels retours, n'hésitez pas à revenir pour vous informer des dernières mises à jour, et à remonter d'éventuel problème de script (ou améliorations).

Bonnes analyses et bon crex à toustes !
